# OA 系統

一個基於 Vue 3 + TypeScript + Element Plus 的 OA 系統。

## 功能特點

### 1. 用戶管理
- 用戶的新增、編輯、刪除
- 角色權限管理
- 用戶狀態控制
- 批量導出功能

### 2. 出缺勤管理
- 排班管理
  - 日曆視圖的排班
  - 批量排班功能
  - 排班表導出
- 出缺勤記錄
  - 考勤統計
  - 出勤率分析
  - 記錄查詢
- 請假審批
  - 請假申請處理
  - 審批流程管理
  - 請假記錄查詢

### 3. 系統設置
- 縣市區域管理
  - 縣市的新增、編輯、刪除
  - 區域的新增、刪除
- 業務人員管理
  - 業務人員資料管理
  - 區域分配
  - 狀態控制
- 課程類型管理
  - 課程類型的新增、編輯、刪除
  - 狀態控制
  - 排序管理
- 系統參數設置
  - 基本設置（系統名稱、Logo等）
  - 郵件設置
  - 自定義參數管理

## 技術棧

- Vue 3
- TypeScript
- Element Plus
- Vue Router
- Pinia
- ECharts
- Vite
