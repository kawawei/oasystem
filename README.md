# Web應用專案

一個類似Google的搜尋應用，整合了OA辦公系統等工具。

## 技術架構

### 前端
- Vue 3
- Vue Router
- Vuex/Pinia
- Element Plus
- Axios

### 後端
- Node.js
- Express.js
- MySQL
- Sequelize
- JWT

## 部署

### 後端部署 (Railway)
1. 創建 Railway 帳號並連接 GitHub
2. 創建新專案並選擇此倉庫
3. 設置環境變量：
   - DB_HOST
   - DB_USER
   - DB_PASSWORD
   - DB_NAME
   - JWT_SECRET
   - NODE_ENV=production
