# Web應用專案

一個類似Google的搜尋應用，整合了OA辦公系統等工具。

## 技術架構

### 前端
- Vue 3
- Vue Router
- Vuex/Pinia
- Element Plus
- Axios

### 後端
- Node.js
- Express.js
- MySQL
- Sequelize
- JWT

## 專案結構

## 部署

### 後端部署
- 創建 Render 帳號並連接 GitHub
- 創建 Web Service
- 設置環境變量：
  - DB_HOST
  - DB_USER
  - DB_PASSWORD
  - DB_NAME
  - JWT_SECRET
  - NODE_ENV=production

## 功能特點

- 用戶認證系統
- 任務管理
- 專案管理
  - 專案階段劃分
  - 進度追蹤
  - 人員分配
- 即時通訊
- 儀表板
