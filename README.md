# Web應用專案

一個類似Google的搜尋應用，整合了OA辦公系統等工具。

## 技術架構

### 核心系統
- 瀏覽器引擎
- 擴展管理器
- API 網關

### 前端
- Vue 3
- Vue Router
- Pinia

## 擴展應用
- OA 辦公系統
  - 任務管理
  - 專案管理
  - 數據分析

## 專案結構

## 部署

### 後端部署
- 創建 Render 帳號並連接 GitHub
- 創建 Web Service
- 設置環境變量：
  - DB_HOST
  - DB_USER
  - DB_PASSWORD
  - DB_NAME
  - JWT_SECRET
  - NODE_ENV=production
